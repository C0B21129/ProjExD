## 負けるな！こうかとん（ex05/fight_kokaton.py）
### ゲーム概要
- ex05/fight_kokaton.pyを実行すると，1600x900のスクリーンに片柳研究所が描画され，
こうかとんと浪人生が戦うゲーム
- こうかとんが退学届けに接触するとゲームオーバーで終了する
- こうかとんが発射する単位を浪人生に当てると浪人生が消える。
### 操作方法
- 矢印キーでこうかとんを左右に移動する
- スペースで単位を発射する
### 追加機能
- 主人公の画像変更
- 連射速度の変化
- スーパーショットの実装
Sキーを押すとスーパーショットが打てる。
スーパーショットは普通の弾よりでかいが、遅く画面上に1つしか存在できない。
- 背景の変更
- 敵の画像変更
